if the 32 bit and 16 bit are both on then it is a function instruction, and if the 8 bit is on, then it is a call instead of a ret


xx111xxx = call
xx110xxx = ret

xx000000 = add
xx000001 = sub
xx000010 = and
xx000011 = or
xx000100 = not
xx000101 = xor
xx000110 = shl
xx000111 = shr
